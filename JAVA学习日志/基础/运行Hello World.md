# 运行Hello World
## 编写Hello World程序
### psvm一键生成 public static void main(String[] args)
```java
public class Hello{
    public static void main(String[] args){
        System.out.print("Hello,World!");
        System.out.print("Who I am?");
    }
}
```
## 找到文件所在位置，在地址前面加入cmd空格
## 进入窗口后，输入javac空格+文件名.java
## 然后输入java空格+文件名回车运行文件
